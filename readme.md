# Domtar Digital Design System
**Version 2019.1.0**

A collection of rules, principles, constraints and best practices, lifted from the brand guidelines and translated into a library of modular web components.

## Components 

- Breadcrumb
- Banner
- Carousel
- Card
- Embed
- Form
- List
- Featured List
- Simple Footer
- Full Footer
- Modal
- Button
- Link
- Table
