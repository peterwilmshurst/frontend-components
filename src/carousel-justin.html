Carousel

The carousel component is a slideshow for cycling through a series of content, built with CSS 3D transforms and a bit of JavaScript. It works with a series of images, text, or other components within this library. It also includes support for previous/next controls and indicators.

The example above incorporates the Banner component but the Card component can also be used (see below).

Code Snippet

    

            <div id="uniqueIdentifier" class="carousel slide" data-ride="carousel">
                <div class="carousel-inner">
                    <div class="carousel-item active">
                        Add slide content here ...
                    </div>
                    <div class="carousel-item">
                        Add slide content here ...
                    </div>
                    <div class="carousel-item">
                        Add slide content here ...
                    </div>
                </div>
                
                <a class="carousel-control-prev" href="#uniqueIdentifier" role="button" data-slide="prev">
                    <span class="carousel-control-prev-icon" aria-hidden="true"></span>
                    <span class="sr-only">Previous</span>
                </a>
                
                <a class="carousel-control-next" href="#uniqueIdentifier" role="button" data-slide="next">
                    <span class="carousel-control-next-icon" aria-hidden="true"></span>
                    <span class="sr-only">Next</span>
                </a>
            </div>
