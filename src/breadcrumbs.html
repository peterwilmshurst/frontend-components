Dependencies

The Breadcrumbs modules uses the font awesome library to render the bullets between each list item.

Options

light/dark backgrounds

By default the Breadcrumbs module adopts the standard link styles set within the theming stylesheet (variables stylesheet).

  • Products
  • Overnight Protection
  • Underwear
  • Attends Premier Underwear

Code Snippet

            
    <div class="breadcrumbs">
        <ul>
            <li><a href="#">Products</a></li>
            <li><a href="#">Overnight Protection</a></li>
            <li><a href="#">Underwear</a></li>
            <li class="current"><a href="#">Attends Premier Underwear</a></li>
        </ul>
    </div>

If the class dark is included within the parent div, the links are styled to work on a dark background. See example and code snippet below.

  • Products
  • Overnight Protection
  • Underwear
  • Attends Premier Underwear

Code Snippet

          
  <div class="breadcrumbs dark">
      <ul>
          <li><a href="#">Products</a></li>
          <li><a href="#">Overnight Protection</a></li>
          <li><a href="#">Underwear</a></li>
          <li class="current"><a href="#">Attends Premier Underwear</a></li>
      </ul>
  </div>
