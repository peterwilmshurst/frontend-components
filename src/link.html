Link

  • Code
  • Usage
  • Style

Links are used primarily as a navigational element. Links may also change what or how data is displayed (i.e., view more, show all).

Links should:

  • Set the expectation of where users will be taken or what action will be performed.
  • Be consistent with the content they point to. For example, if a navigational link leads to a page called Frequently asked questions, label the link “FAQ’s”.
  • Use descriptive text and avoid using generic content such as “click here”.
