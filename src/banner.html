Options

Image

Images are added and saved as background images within the css. A desktop, tablet and mobile version of each image should be provided. The css below will ensure the correct image is shown on the relevant device. Retina versions of each image should also be provided, these will be displayed on higher resolution displays.

                        

                .custom-img-1 {
                    background-image: url('../img/banner/demo_banner-01-mobile.jpg');
                    background-image: -webkit-image-set(url(../img/banner/demo_banner-01-mobile.jpg) 1x, url(../img/banner/demo_banner-01-mobile@2x.jpg) 2x);
                
                    @include media-breakpoint-up(sm) {
                        background-image: url('../img/banner/demo_banner-01-tablet.jpg');
                        background-image: -webkit-image-set(url(../img/banner/demo_banner-01-tablet.jpg) 1x, url(../img/banner/demo_banner-01-tablet@2x.jpg) 2x);
                    }
                
                    @include media-breakpoint-up(md) {
                        background-image: url('../img/banner/demo_banner-01-desktop.jpg');
                        background-image: -webkit-image-set(url(../img/banner/demo_banner-01-desktop.jpg) 1x, url(../img/banner/demo_banner-01-desktop@2x.jpg) 2x);
                    }
                }

Position

The textbox, if included, can be positioned by amending this class. The options are listed below. Simply remove the content-area div if no text/content is needed.
Please note: By default, content area will align vertically bottom on a mobile device unless the mobile-top class is used.

Class Description Example
left-33 33% width of parent container, left-hand side Link
left-50 50% width of parent container, left-hand side Link
center-33 33% width of parent container, center Link
center-50 50% width of parent container, center Link
center-100 100% width of parent container, center Link
right-33 33% width of parent container, right-hand side Link
right-50 50% width of parent container, right-hand side Link

Background Colour

Options are listed below:

blue-bg Blue background Link
