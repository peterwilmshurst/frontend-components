Carousel

  • Code
  • Usage
  • Style

The carousel component is a slideshow for cycling through a series of content, built with CSS 3D transforms and a bit of JavaScript. It works with a series of images, text, or other components within this library. It also includes support for previous/next controls and indicators.

The example above incorporates the Banner component but other components can also be used.
